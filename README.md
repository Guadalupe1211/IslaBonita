## Descripción

Este proyecto consiste en la creación de un entorno natural jugable en Unity, diseñado para servir como base para futuras mecánicas de gameplay y exploración. 
El escenario incluye montañas, vegetación, cuerpos de agua e iluminación ambiental, buscando crear un espacio visualmente atractivo y optimizado 
para la navegación del jugador.
El objetivo principal fue aprender y aplicar las herramientas de creación de terrenos de Unity, así como técnicas básicas de diseño de niveles, 
ambientación y optimización.

## Características Implementadas

* Terreno 3D generado con Unity Terrain.
* Esculpido de montañas, valles y zonas transitables.
* Sistema de texturizado mediante capas de:
  * Pasto
  * Tierra
  * Roca
* Lago integrado al entorno.
* Vegetación distribuida con árboles y pasto.
* Skybox configurado para mejorar la atmósfera visual.
* Iluminación básica y sombras.
* Terrain Collider para navegación del jugador.
* Escenario optimizado para pruebas de gameplay.

## Tecnologías Utilizadas

* Unity
* Terrain Tools
* Universal Render Pipeline (URP)
* C#
* Sistema de iluminación de Unity
* Terrain Collider
